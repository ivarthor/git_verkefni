Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd   vísar á möppu
	cd ~ vísar á ~
	ls   gefur upplýsingar
	pwd  sýnir hvaða drifi forritið er að vinna í
	mkdir gerir directory
	

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone   gerir leið fyrir þig til að vinna í repository
	git log     sýnir login yfir því hverju þú ert búinn að committa
	git status  sýnir þér skjölin sem þú ert búinn að breyta og þeim sem að þú átt eftir að breyta
	git diff    sýnir breytingar
	git checkout leyfir þér að breyta mistökum

	
3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff                 	sýnir breytingar
	b) git diff --staged 			sýnir breytingar eftir git add
	c) git diff commit1 commit2 	sýnir munin á milli commit1 og commit2


4. 	Hvað er útgáfustýring (e.version control)?  Stýrir breytingum á skjölum  og stórum vefsíðum


5.	Hverjir eru helstu kostir við að nota GIT?  Það er hægt að léttilega gera verkefni með öðrum.


6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?  Eins oft og þarf, maður er svo oft að breyta skjölum að það er kki hægt að segja hversu oft maður mun committa yfir allt vetrkefnið

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?   "Working directory" í git, er directory sem heldur öllum skjölunum áður en þau eru breytt. "Staging area" er staðurinn sem er geymt hvað mun fara  þegar þú gerir commit. "repository" er þar sem er geymt öll skjölin eftir að þú ert búinn að committa þau


8.	Hvenær er sniðugt að nota greinar (branches)? Þegar það eru margir að vinna á einu verkefni en samt öðruvísi parta af þessu verkefni, þá er hægt að branch-a út og splitta aðeins verkefninu upp, síðan er hægt að merge-a það saman seinna. 

9.	Hver fann upp GIT?    Linus Torvalds, hann gerði líka linux

10.	Hvar er GIT helst notað?   í forritun og sem geymslu forrit
